<h2>Project : Online Market Catalog</h2>

<p>This project consists of Online Market Stores and List and of items that are available in the stores.<p>

<p>This project is implemented using Pep8 guidelines.(To check the pep8 guide line please run using pycodestyle --first filename.py)</p>
<p>You can install, upgrade, and uninstall pycodestyle.py with these commands:(If you are using virtual envi please use sudo before pip)</p>
<ul>
<li>pip install pycodestyle</li>
<li>pip install --upgrade pycodestyle</li>
<li>pip uninstall pycodestyle</li>
</ul>

<p>The unauthorized users can view the publicstores and public items but cannot perform CRUD operation.</p>

<p>To perform CRUD operations the user must have to login using google signin services.</p>

<p>To run the project please run the project.py which is available inside fullstack folder.</p>

<p>after running the program please visit localhost:5000/store.</p>

<p>I have used vagrant 2.0.1 and virtual box 5.1 to complete the project.</p>

<p>After installing vagrant to connect to virtual box use following commands.</p>
<ul>
<li>To initialize vagrant use ------> vagrant init</li>
<li>To start vagrant machine use -------> vagrant up</li>
<li>To establish connection to virtual machine use --------->vagrant ssh</li>
<li>cd/vagrant</li>
</ul>

<p>I have used flask, google oauth2 and SQLAlchemy to complete the project.</p>

<p>Following commands to be installed successfully run this project are.(If you are using virtual envi please use sudo before pip)</p>
<ul>
<li>pip install SQLAlchemy</li>
<li>pip install Flask</li>
<li>pip install --upgrade google-api-python-client</li>
</ul>

<p>To use your own authentication services please visit the following links and read the available instructions.</p>
